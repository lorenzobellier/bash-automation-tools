# Bash Automation Tools

This repository contains Bash scripts for Linux system automation, monitoring, and basic security tasks.

## Features
- Backup user home directories
- Monitor login attempts
- Check status of key services
- Simple alert for suspicious activity

## Learning goals
- Automate repetitive tasks
- Monitor system activity
- Practice scripting for security purposes
- Prepare practical projects for CV

## Usage
1. Make scripts executable: chmod +x scripts/*.sh
2. Run scripts as root when necessary
