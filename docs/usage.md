# Usage Guide

## Scripts

1. backup_users.sh - backups of /home directories
2. monitor_logins.sh - display last successful and failed logins
3. check_services.sh - check status of SSH, Apache, MySQL
4. alert_script.sh - alerts if suspicious root login attempts detected

## Requirements
- Ubuntu Server
- Root access
- Bash

## Learning Outcome
- Gain automation skills
- Improve server monitoring
- Prepare for real-life sysadmin and security tasks
